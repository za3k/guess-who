**Guess Who?** was made by [Zachary "za3k" Vance](https://za3k.com)" Vance as part of [Hack-a-Day](https://za3k.com/hackaday) a challenge to do one new project each day in November. 

Play [online now](https://za3k.github.io/guess-who) with a friend. You will each need your own screen to look at.

It is based on the game of the same name by [Hasbro](https://shop.hasbro.com/).

Faces are taken from [thispersondoesnotexist.com]
